# Лабораторная №2

## Структура проекта

**1.** `CMakeLists.txt` — CMake-скрипт сборки проекта  

**2.** `main.cpp` — тестовая программа для проверки библиотеки  

**3.** `lib/process_lib.h` — заголовочный файл библиотеки  

**4.** `lib/process_lib_win.cpp` — реализация библиотеки для Windows  

**5.** `lib/process_lib_posix.cpp` — реализация библиотеки для Unix/POSIX  

**6.** `scripts/build.sh` — скрипт сборки для Linux/macOS/WSL  

**7.** `scripts/build.cmd` — скрипт сборки для Windows  

## Функции библиотеки
**-** `start_process`: запускает процесс  

**-** `wait_process`: ждёт завершения процесса и возвращает код выхода 

**-** `free_process`: освобождает ресурсы

## Тестовая программа main.cpp проверяет

**-** простую команду (echo Hello)  

**-** длительную команду (timeout / sleep) 

**-** листинг директории (dir / ls)

**-** несколько команд подряд

**-** некорректную команду
