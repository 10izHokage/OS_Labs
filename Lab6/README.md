# Лабораторная №6

## Структура проекта

**1.** CMakeLists.txt — скрипт сборки проекта

**2.** src/main.cpp — графическое приложение (GUI) для отображения температуры

**3.** src/TemperatureGraph.h / TemperatureGraph.cpp — компонент для отрисовки графика температуры

**4.** src/db.h / db.cpp — работа с базой данных SQLite

**5.** src/serial_port.h — интерфейс получения данных

**6.** src/serial_port_win.cpp — реализация для Windows

**7.** src/serial_port_posix.cpp — реализация для Linux

**8.** data/temperature.db — база данных

**9.** D_example/H_example - примеры работы (на Windows)

## main.cpp

- создаёт графическое окно с помощью SFML 3

- подключается к серверу через HTTPClient

- получает данные о температуре через HTTP API: `/hourly /daily`

- передаёт данные в TemperatureGraph

- отображает график температуры с подписями осей и сеткой

- позволяет переключать режимы отображения:

`Hourly` — почасовые данные

`Daily` — суточные данные

- отображает подписи: название приложения, текущий режим отображения

## TemperatureGraph

Компонент для визуализации температуры. Поддерживает:

- отрисовку линии графика

- сетку

- подписи осей

- автоматическое масштабирование по данным

- отрисовку графика в заданной области окна SFML

## GUI

- окно создаётся размером 1200×700 пикселей

- шрифт для всех подписей: Arial

- линии графика отображаются зелёным цветом, сетка и подписи — белым

- переключение между режимами Hourly и Daily осуществляется через клавиши H и D

- график и подписи обновляются при каждом кадре
